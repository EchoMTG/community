# EchoMTG Community

Welcome to the community hub for [EchoMTG](https://www.echomtg.com) — the collection tracker and value-watching tool for Magic: The Gathering (and a few other games).

This repo is the place to:

- **Report bugs** — something's broken, a price is wrong, a page won't load
- **Request features** — what would make EchoMTG better for you
- **Start a discussion** — share strategies, ask about the API, help other users

## Quick links

- **Report a bug →** [New bug report](../../issues/new?template=bug_report.yml)
- **Request a feature →** [New feature request](../../issues/new?template=feature_request.yml)
- **Ask a question / start a discussion →** [Discussions tab](../../discussions)

## Before you post

- Search open and closed issues first — someone may have already raised it
- For bugs: include what you did, what you expected, what actually happened, and a screenshot if the UI is involved
- For API questions: check [echomtg.com/api](https://www.echomtg.com/api) first — the docs there cover auth, endpoints, and common shapes
